# UI Spec (MVP)

Style: clean SaaS admin, neutral colors, lots of whitespace.
Layout: centered container, max-width 1000px, consistent header.

Components:
- Button: primary + secondary
- Form: labels above inputs, full width inputs, inline errors
- Table: zebra rows, subtle borders, status badges
- Flash: success/info/error banners

No custom design per page; reuse components.
